Ce este curățarea cu gheață carbonică

Gheața carbonică (CO₂ solid) este proiectată controlat pe suprafață. La impact, murdăria este desprinsă, iar gheața sublimează direct în gaz, fără a lăsa reziduuri secundare.

De ce este ideală pentru motoare

Metoda este folosită când este necesară curățare eficientă fără abraziune, fără apă și fără a afecta suprafețe sensibile sau componente de precizie.
